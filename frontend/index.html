Austria First Transcripts

Was ist das? Automatische Verschriftlichung des FPÖ-Radiosenders "Austria First". Archiv-Sendungen stammen von der offiziellen Website, Live-Aufnahmen werden direkt vom Stream mitgeschnitten.

Wie funktioniert das? Die Texte werden von Whisper erstellt, einer Software zur automatischen Spracherkennung. Mehr dazu auf Wikipedia.

Wie genau ist das? Bei Namen, Fachbegriffen oder undeutlicher Sprache können Fehler auftreten. Live-Aufnahmen enthalten auch Musik. Dabei kann es passieren, dass Songtexte transkribiert werden oder die Software unsinnigen Text erzeugt. Die Original-Audiodateien stehen zum Vergleich bereit.
